# audio-kit-arduino-examples
音頻擴展板 Arduino專案 範例程式碼
# Arduino所支援的開發板
安裝Arduino IDE，配合開發板管理員進行開發板安裝，本範例包括:

  Arduino UNO開發板。

  STM32F030開發板，STM32F072，STM32F103，STM32F401。

  ESP8266開發板，ESP8266-12E，ESP8266-12F。

  ESP32開發板

  ESP32-C3開發板
